# 🛒 Mini Supermarket Management System

A simple **Mini Supermarket Management System** designed to help manage products, sales, inventory, and basic store operations.
This project is suitable for learning **web development, database management, and CRUD operations**.

---

## 📌 Project Overview

The Mini Supermarket system provides basic features for managing a small retail store, including:

* Product management
* Category management
* Inventory tracking
* Sales management
* User management (admin/staff)
* Dashboard for store overview

This project demonstrates how a **basic POS-like system** can be implemented for small businesses.

---

## ⚙️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / Apache

## 🚀 Installation Guide

### 1️⃣ Clone the repository

```
git clone https://github.com/nhphuc6524-dotcom/mini-market.git
```

### 2️⃣ Move project to XAMPP

Place the folder inside:

```
xampp/htdocs/
```

Example:

```
xampp/htdocs/mini-supermarket
```

---

### 3️⃣ Import Database

1. Open **phpMyAdmin**
2. Create a new database

```
mini_supermarket
```

3. Import the SQL file located in:

```
database/mini_supermarket.sql
```

---

### 4️⃣ Run the project

Open browser and access:

```
http://localhost/mini-supermarket
```
